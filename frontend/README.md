# Angular Front

install Angular CLI:
```
npm install -g @angular/cli
ng version
```

check the version of an installed npm global packages:
```
npm list -g
```

create new project:
```
ng new
```

Install Bootstrap:  
```
npm install bootstrap@5.3.3
```

generate:
```
ng generate component
ng generate module
ng generate service
```

run the application
```
cd angular-app
ng serve --open
```
